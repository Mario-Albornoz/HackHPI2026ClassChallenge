To run the evaluation for this project you can run the File under model called TrainYologenerater.py

Requirements:
python3.11 (otherwise dependicies might fail)

```
python3.11 -m venv venv
source venv/bin/activate
pip install -r << requirements.txt
python3 model/trainYoloGenerated.py
```

within this file you can change the parameter test to True or False based on if you want to train or evualuate the model

All datasets are confidential and therefore not provided in this repository.
